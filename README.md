# Proceso de aprendizaje autodidacta en Node.js

Este repositorio fue creado con el objetivo de **practicar y reforzar conocimientos de Node.js**, comenzando desde conceptos básicos hasta el uso de **Express** y el trabajo con **bases de datos relacionales**.

El proyecto está organizado por **fases**, donde cada una representa un avance progresivo en el aprendizaje. A lo largo de estas fases se abordan temas como:

- Fundamentos de **Node.js**.
- Manejo de archivos y operaciones **asíncronas**.
- Creación de **servidores** y **rutas** con Express.
- Implementación de la arquitectura **MVC**.
- Uso de **bases de datos** y relaciones entre tablas.
- Validaciones, middlewares y manejo de errores.
- Consultas SQL y uso de **JOIN** para obtener información relacionada.

Este repositorio **no está pensado para producción**, sino como un espacio de **práctica y estudio**, donde se experimenta con distintos conceptos y estructuras propias del desarrollo backend con JavaScript.

Su finalidad principal es servir como:
- Registro del proceso de aprendizaje.
- Referencia personal para futuros proyectos.
- Apoyo para reforzar conceptos de backend con Node.js.
