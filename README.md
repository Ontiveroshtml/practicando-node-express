# Proyecto de aprendizaje en Node.js

Este repositorio fue creado con el objetivo de **practicar y reforzar conocimientos de Node.js**, comenzando desde conceptos básicos hasta el uso de frameworks como **Express**.

El proyecto está organizado por fases, donde cada una representa un avance progresivo en el aprendizaje, abordando temas como:
- Fundamentos de Node.js.
- Manejo de archivos y operaciones asíncronas.
- Creación de servidores y rutas con Express.
- Organización del código y conexión con bases de datos.

Este repositorio no está orientado a un entorno de producción, sino que funciona como un **espacio de práctica y estudio**, permitiendo experimentar con diferentes conceptos y estructuras propias del desarrollo backend con JavaScript.

Su finalidad principal es servir como referencia personal y como registro del proceso de aprendizaje en Node.js.
