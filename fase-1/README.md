## FASE 1

En esta fase se practicó **Node.js puro**, aprendiendo los fundamentos del entorno.

Se estudiaron los siguientes conceptos:
- Acceso a **rutas** y organización de archivos.
- **Exportación de módulos**.
- Uso de `process.argv` para recibir argumentos desde la terminal.
- Conceptos básicos del módulo **File System (`fs`)** para **leer y crear archivos**.

El objetivo fue comprender la estructura básica de un proyecto en Node.js y su interacción con el sistema de archivos.
