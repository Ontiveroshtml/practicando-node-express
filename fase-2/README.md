## FASE 2

En esta fase se practicó **Node.js puro**, enfocándose en el uso de **promesas** con el módulo `fs`.

Se implementaron ejercicios para:
- Crear un **logger** utilizando `setInterval`.
- Construir un **timer** combinando `setInterval` y `setTimeout`.
- **Leer y copiar archivos** mediante el uso de **streams**.

El objetivo fue comprender el manejo de tareas asíncronas y el trabajo con el sistema de archivos en Node.js.
