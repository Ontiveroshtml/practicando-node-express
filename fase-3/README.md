## FASE 3

En esta fase se aprendió el uso de **Node.js con Express**, practicando la **inicialización del servidor** y la **creación de rutas**.

También se implementó lógica básica dentro de las rutas para **insertar, editar, actualizar y eliminar datos en memoria**, con el objetivo de comprender el funcionamiento de las peticiones HTTP.

Además, se estudió el uso de los **middlewares** y su papel dentro del flujo de solicitud–respuesta.
